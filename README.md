# Medieval Survivor — 中世紀生存塔防遊戲

> 以 HTML + Phaser 3 製作的生存塔防遊戲。玩家扮演村莊守護者，在開放世界中採集資源、建造與升級建築、指揮友方單位，並在波次攻擊中抵禦強盜外敵。

---

## 遊戲概念

- **類吸血鬼生存者**：俯視視角，自動攻擊，波次敵人
- **資源採集**：在地圖上砍樹、採石，或透過採集所自動採集
- **建築系統**：10 種建築，多段升級，含兵營與法師塔
- **友方單位**：兵營召喚近戰/遠程士兵，法師塔召喚友方法師（AoE）
- **集結指令**：`[F]` 鍵切換士兵/法師追隨玩家或駐守建築
- **波次系統**：白天採集建設，夜晚抵禦敵人；波次越後難度越高
- **平衡設定**：主選單可調整 20 項平衡數值，金幣可換算資源

## 波次設計

| 波次類型 | 敵人組合 |
|---------|---------|
| 白天1 (Wave 1,4,7…) | 強盜 100% |
| 白天2 (Wave 2,5,8…) | 強盜 60% + 弓箭手 40%（Wave 9+ 加法師 12%） |
| 夜晚 (Wave 3,6,9…) | 強盜 45% + 弓箭手 30% + 重甲 25%（Wave 9+ 加法師 20%） |

每波敵人數：`max(1, round((3 + wave×3) × SPAWN_MULT))`。夜晚 ×1.8。

## 操作方式

| 操作 | 功能 |
|------|------|
| WASD / 方向鍵 | 移動玩家 |
| 靠近資源節點（90px）停留 0.8 秒 | 自動採集（進度條顯示） |
| `[B]` | 開啟/關閉建造選單 |
| 建造選單 → 選擇 → 左鍵點地圖 | 放置建築（綠色半透明預覽） |
| `[ESC]` / 右鍵 | 取消建築放置 |
| 點擊已放置建築 | 開啟升級面板 |
| `[P]` | 暫停/繼續 |
| `[F]` | 切換友方單位集結模式（跟隨玩家 / 返回建築駐守） |

## 平衡設定系統

主選單「⚙ 平衡設定」按鈕可在遊戲開始前調整 20 項數值：
- 玩家 HP / 速度 / 攻擊力 / 射程 / 攻擊間隔
- 各敵人 HP / 速度 / 射程
- 波次準備時間 / 間隔 / 生成倍率
- 資源重生時間 / 木材石材獲取量
- 金幣兌換匯率（1金=N木材 / 1金=N石材）

黃色粗體 = 已修改自預設值。GameOver 畫面顯示本局設定。

## 開發日誌

### 2026-03-26 — Phase 5 + 6 + 額外功能

**Phase 5 — 自動化建築：**
- **採集所**：建築 130px 範圍內的資源節點每 800ms 自動採集一次，玩家無需手動靠近
- **維修工**：每 3 秒掃描所有建築，找出最低 HP% 的目標，以 8HP/s 修復，顯示浮字 `+8`

**額外功能（平衡設定 + 暫停 + 金幣兌換）：**
- **P 鍵暫停**：凍結物理引擎，所有 AI / 波次 / 採集全部暫停，顯示 PAUSED 覆蓋層
- **20 項平衡設定面板**：主選單可調整玩家/敵人/波次/資源/匯率共 20 項數值，GameOver 顯示本局設定（黃=已修改）
- **金幣兌換資源**：建造/升級資源不足時自動以金幣補足差額，匯率可在平衡面板設定
- **弓箭手 AI 修正**：弓箭手改為永遠優先追蹤玩家，建築傷害降至 3（原 10）

**Phase 6 — 兵營系統：**
- **Barracks 兵營**：定時召喚士兵（lv1: 2名近戰，lv2: 4名混合），可升至 Lv2
- **Soldier 士兵**：近戰型（Bandit 行為，80HP）或遠程型（Archer 行為，50HP），繫繩至兵營 200px，遇敵主動交戰
- **敵人識別士兵**：修正 atkRange 計算（物理碰撞使 body 距離 ~32px，改為 40px 才能觸發攻擊）
- **`[F]` 集結模式**：切換繫繩點從建築到玩家，所有友方單位跟隨玩家行動

**法師塔（新建築）：**
- **MageTower 法師塔**：定時召喚友方法師（lv1: 1名，lv2: 2名），邏輯與兵營完全相同
- **AlliedMage 友方法師**：AoE 爆炸（半徑 60px），**無友傷**（只傷害敵方），青綠色爆炸特效；繫繩 200px；同樣受 `[F]` 集結模式影響
- 所有敵人（含弓箭手）會識別並優先攻擊友方法師
