## Phase 7 完成內容

### Feature 1 — 音效系統 ✓
- 新增 `src/utils/SoundManager.js`：Web Audio API 程式生成音效，無需外部音頻檔案
- 支援 8 種音效類型：玩家射擊、敵人受傷、敵人死亡、建築碰撞、爆炸、資源採集、波次警報、新波次號角
- 各音效有冷卻時間防止重疊音爆
- 整合點：GameScene（射擊/採集/爆炸/建築碰撞）、Enemy（受傷/死亡）、WaveSystem（波次開始）

### Feature 2 — 視覺特效強化 ✓
- **敵人死亡粒子**：6 顆紅色小點從死亡位置向外散射（Enemy._die）
- **建築損毀冒煙**：HP < 30% 的建築每 900ms 冒出一縷上升灰煙（GameScene._updateBuildingSmoke）
- **採集浮動文字**：玩家採集到木材/石材/糧食時，飄出 +木 / +石 / +食 彩色文字（GameScene._showCollectFloat）

### Feature 3 — 存檔系統 ✓
- 已於先前版本完成：localStorage `medieval_survivor_save` 記錄最高波次
- GameOverScene 顯示「新紀錄！」或當前最高記錄
- MenuScene 顯示歷史最高波次

### Feature 4 — 平衡調整 ✓
- 友方法師召喚速率：12000ms → 10000ms（法師出現更頻繁）
- 採集所採集範圍：130px → 150px（涵蓋範圍更廣）
